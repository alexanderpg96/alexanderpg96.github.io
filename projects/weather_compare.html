Weather Compare

Weather Compare is a personal project that I put together because I travel frequently around the United States.

With this web application, you can compare two locations and the difference in temperature between them

This project was one of my first ever, and was written in pure HTML, CSS, and JS. Weather data is retrieved via API calls to WeatherUnderground.
